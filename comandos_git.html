comados de configuração

> comandos quando istalamos o git em um computador

  • git config --global user.email"coloque@seu.email": ela configura seu email.
  • git config --global user.name"": ela configura seu nome.

> tipos de git: Gerenciador de código fonte

  • git bash: terminal do git onde utilizamos os comandos git.
  • git init: informa que a pasta passará a ser monitorada pelo git, para controlar as modificações realizada
  • git status: verifica se há alguma modificação na pasta do projeto.
  • git add "nome_do_arquivo": insere os arquivos para serem versionados (criar uma nova versão).
  • git branch -m "novo_nome":altera o nome da branch para um novo_nome
  • git commit -m "uma descrição do que foi feito" : cria uma versão dos arquivos adicionados com co comando git add
  • - commit : é a criação de uma versão de um ou mais arquivos, com uma descrição do que foi feito.
  • - git add cria uma versão dos arquivos adicionados com co comando git add
  • - branch: é a linha do tempo de um determinado código. Normalmente tem-se uma linha principal e outras ramificações para desenvolvimento. No git a linha principal ela tem o nome de master ou main.
